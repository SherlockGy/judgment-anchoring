# 领域:中文小说创作

**适用:** 中文武侠、故事、叙事散文。

## 锚点配方

| 维度 | 人名 | facet |
|---|---|---|
| 叙事节奏 | 古龙 | 短句(含单句段)制造速度,沉默和留白多于解释;一个人的危险不用招式描写,用在场者的反应来写 |
| 意境与场景 | 温瑞安 | 场景不是背景板,场景就是情绪本身;光线、湿度、人声密度、环境都在叙述而非装饰 |
| 人物正邪边界 | 金庸 | 真正的高手通常是最不显眼的那一个;人物有不可解的灰度,最好的角色是说不清他是好人还是坏人的那种 |

## 必要规则

- 全文用中文写作,不夹杂英文词汇或拼音
- 不使用网络流行语、现代俚语、科技术语
- 不使用武侠套话(剑眉星目、杀气腾腾、空气凝固等)
